# Préférences de Développement & Design 🎨

Guide des choix esthétiques et conventions préférées.

- **Interface Utilisateur** : Utilisation de la police **Outfit** (Google Fonts). Design "Glassmorphism" avec flous et bordures subtiles.
- **Langue** : L'interface (UI) est en **Français**. Le code (variables, classes) est en Anglais. Les documentations sont bilingues ou en Français pour l'utilisateur final.
- **Efficacité Contextuelle** : Les fichiers d'instructions IA doivent rester courts (max 150 lignes).
- **Navigation** : Le point d'entrée après login est toujours `setup.php` pour permettre une personnalisation rapide.
- **Composants** : Pas de frameworks lourds. Vanilla CSS et PHP pur.
